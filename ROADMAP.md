# Eurion OS Roadmap (Draft)

## v0.0.1 — VM Demo Alpha
Obiettivo: demo ripetibile, consumer-first, “non sembra una distro”.
- Desktop KDE neo-futuristico (tema + coerenza UI)
- Eurion Hub v0: catalogo curato 20–30 app essenziali
- Aggiornamenti atomici + rollback dimostrabili
- Compatibilità Windows v0: wizard “Esegui App Windows”
- Privacy Center: telemetria OFF by default + principi trasparenza

## v0.1 — Private Beta (early adopters)
- ISO Eurion ufficiale
- Installer Eurion (nessun terminale)
- Control Center v1 (Privacy & Security)
- Compatibilità Windows migliorata: profili app ufficio
- Driver e periferiche: checklist stampa, bluetooth, audio

## v0.3 — Public Beta
- “Eurion Certified Hardware” (lista device testati)
- Update firmati end-to-end
- Store Eurion più maturo (rating, curated list)
- Gaming: Steam + layer compatibilità migliorato

## v1.0 — Release
- UX finale
- stabilità e performance
- supporto commerciale opzionale
- governance europea formalizzata
