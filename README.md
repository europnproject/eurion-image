# Eurion OS (alpha)

Eurion OS è un sistema operativo consumer-first con identità europea:
- **Secure-by-design**
- **GDPR-first**
- Telemetria **trasparente e controllabile**
- Base **immutabile** con aggiornamenti atomici e rollback
- App in sandbox (store-first)
- Compatibilità Windows: requisito di prodotto (UX guidata)

## Stato
Alpha: stiamo costruendo la prima demo ripetibile in VM.

## Principi non negoziabili
1) **User-first**: nessun terminale per l’utente finale.
2) **Trasparenza**: l’utente può vedere cosa viene raccolto/inviato.
3) **Sicurezza by default**: hardening e isolamento di base.
4) **Aggiornamenti affidabili**: atomici + rollback sempre possibile.
5) **Compatibilità reale**: file office, stampanti, gaming, app Windows quando necessario.
